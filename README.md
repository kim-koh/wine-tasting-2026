# Wine Tasting Event 2026

Welcome to the Wine Tasting Event 2026 website! This application is designed to streamline RSVP management and enhance the guest experience during the event.

## Features

- **RSVP Management:** Guests can easily confirm their attendance and update their details.
- **Wine Catalog:** Browse detailed information about each wine featured at the event.
- **Mobile Friendly:** Optimized for use on smartphones and tablets during the event.

## Usage

1. **RSVP:** Visit the RSVP page to confirm your attendance.
2. **Browse Wines:** During the event, use the app to explore tasting notes, origins, and pairings for each wine.

## Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/wine-tasting-2026.git
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start the development server:
    ```bash
    npm start
    ```
