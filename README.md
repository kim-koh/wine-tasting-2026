# Wine Tasting Event 2026

Kitschy wine app for my annual wine tasting event

## Features

- **Wine Catalog:** Browse detailed information about each wine featured at the event.
- **Wine Rating:** Guests can rate and take notes on the wines as they taste.
- **Dietary Information:** Guests can view important dietary and allergen information about the other food and drinks at the event. 
- **Mobile Friendly:** Optimized for use on smartphones and tablets during the event.
- **Email summary:** Guests can email themselves a summary of the wines and their notes after the event.
- **Hidden admin page:** Hosts of the event can test and manage the site on a page hidden from the rest of the users.

## Usage

1. **RSVP:** Visit the RSVP page to confirm your attendance.
2. **Browse Wines:** During the event, use the app to explore tasting notes, origins, and pairings for each wine.

## Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/wine-tasting-2026.git
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start the development server:
    ```bash
    npm start
    ```
